Limitation in the Apple iCloud - 18 names of files and folders that cannot be used in iCloud

Nabeel Shoro February 6, 2020 icloud
Dropbox, OneDrive and tmp are not a good idea while you sync your data to iCloud on iPhone.
No matter how hard you try, you won’t be able to sync a folder called Dropbox with iCloud . It turned out that this is not the only limitation in the Apple iCloud cloud service.
In this article, we will clarify that you can create the folder or file as the following name list of folder and file but it will not be synchronized with other devices. On a Mac, it will display “Inappropriate” with no sync icon. Here is a complete list of invalid file names in iCloud.

Invalid File and Folder Names in iCloud.

  • strictly.DS_Store
  • begin with (A Document Being Saved
  • contains .nosync(in any case)
  • strictly .ubd
  • contains .weakpkg
  • strictly tmp (in any register)
  • strictly .tmp (in any register)
  • strictly desktop.ini (in any register)
  • begin with ~$
  • strictly Microsoft User Data (in any register)
  • strictly $RECYCLE.BIN (in any register)
  • strictly iPhoto Library (in any register)
  • strictly Dropbox (in any register)
  • strictly OneDrive (in any register)
  • strictly IDrive-Sync (in any register)
  • strictly .dropbox (in any register)
  • strictly .dropbox.attr (in any register)
  • strictly icon\r (in any register)

Invalid Extensions (in any case) in iCloud.

  • tmp
  • photoslibrary
  • photolibrary
  • aplibrary
  • migratedaplibrary
  • migratedphotolibrary
  • migratedaperturelibrary

Why do you need this restriction?

It seems that users do not copy the root folders of Dropbox or other services and add them to iCloud Drive along with all hidden files that will conflict with iCloud. As a result, two cloud services will “fight” for the same files, which can lead to data loss or the creation of unnecessary copies.
