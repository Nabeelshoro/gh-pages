WhatsApp will stop working in 2020 - Complete List of Smartphones.

Nabeel Shoro December 16, 2019 Smartphones

The developers of the WhatsApp messenger in the year 2020 refuse to support old mobile operating systems. In this article, we will guide you on which smartphones will not support WhatsApp? if you are still using bellow listed Smartphones then it is confirmed that your WhatsApp will not work and it might be banned.

The process of banning the old operating system smartphones has two stages. First, starting January 1, 2020, WhatsApp will stop working on all Windows Phone devices. And Second, from February 1, 2020, the WhatsApp messenger will not work on smartphones with Android 2.3.7 and iOS 8.

Here is the Complete list of Smartphones that will not be supported by WhatsApp in the year 2020.

  • Windows phone

Starting January 2020, WhatsApp refuses all smartphones with Windows Phone operating system - up to the latest 8.1. These are almost all smartphones until the 2014 release. These are mainly Lumia smartphones, which were manufactured by Nokia in partnership with Microsoft.

Popular smartphones on Windows Phone that will lose WhatsApp are Lumia 520, 525, 620, 625, 630, 720, 820, 920, 925, 1020, 1320. For some smartphones from this list, it is possible to install a test or unofficial build of Windows 10 Mobile, but often this firmware is unstable. 

But WhatsApp will continue to work on all smartphones that have upgraded to Windows 10 Mobile. But Microsoft itself has already closed support for this operating system, so the messenger on these smartphones will last at best a couple more years.

  • iPhone

WhatsApp will not work in iPhone 4 and older models. That is all Apple smartphones that came out before 2011.

If iPhone 4S, for example, received an update to iOS 9, so WhatsApp will still work on it. As with all newer Apple smartphones.

  • Android

From February 2020, WhatsApp will no longer support all smartphones on Android 2.3.7 and older. Dozens of models that came out between 2009-2011 will be banned. Now Android 2.3 and its variations occupy a share of 0.3% of all active smartphones on Android - taking into account the total number of devices, it is about 7.5 million smartphones worldwide.

list of Andriod smartphones which will be banned of WhatsApp:

WhatsApp will stop working on the first Samsung Galaxy S, Galaxy Mini, Galaxy Gio, Google Nexus One, Sony Ericsson Xperia X10, Sony Ericsson Xperia Play, LG Optimus One, HTC Wildfire S, HTC Desire, HTC Desire HD, HTC Desire Z and many other smartphones that have not updated to Android 4.0 Ice Cream Sandwich.

How to use WhatsApp in old smartphones?

So, what should we do next? Yes, the answer is we can use WhatsApp in old Smartphones if we install unofficial firmware of Android 4.0 or newer. but hurry up install it before February 2020.

To simplify, WhatsApp support will lose the old smartphones that came out before 2011.
