Windows 7 Trick | How to Setup Local Host with WordPress

Nabeel Shoro August 30, 2014 Computer Tips and Tricks

Your computer is not only your personal computer but you can use it for your personal local host and it work as same as web host, where you can experience web development environment with Apache2, PHP, MySQL database and PhpMyAdmin which allows you to manage your databases as same as web host do. You can also install Content Management Software like WordPress etc.

As you know that WordPress is the very popular Content Management System in blogging. WordPress has many advantages and reasons for use. You can also install WordPress in your personal computer for the purpose of learning and testing

So I am going to show you some basic information of installing of local host in you computer.

Why to use local host and WordPress in personal Computer?

In many cases, the designers preferred to install WordPress on a local host to test the utility programs and make changes to the site before being presented to the public.

Note: If you install Wampserver locally on your computer, then the only person who can see this site is you. Wampserver 2.5 Does not support Windows Xp, Windows Server 2003.

What is WAMP?

Wamp or Wampserver gives all possibilities of local host to a computer running Windows. You need to prepare a Wampserver environment and install your own WordPress on Windows. It's free and easy to use.

How to Install Wampserver on your PC

The first thing you need to do is go to Wampserver site http://www.wampserver.com/en/ and download the latest version of Wampserver. Will be offered different options you have on the download page. Simply select the option that applies to the Windows system according you computer configuration (64-bit or 32-bit).

Once you download Wampserver, run the installation program and follow the instructions that appear on the screen. During installation, you will ask for installation place. By default it will use the Internet Explorer browser, but you can change it to Google Chrome or Firefox by locating the browser in Program Files. Once the installation is finished, run the Wampserver.

Create a Database for WordPress

Wampserver comes with the application of phpMyAdmin a web application to manage MySQL databases. Click on the icon in the task bar of Wampserver Windows, and then click on phpMyAdmin.

PhpMyAdmin will open in a new window. Click on Databases in PhpMyAdmin to create a new database for the WordPress. Choose a name for your database and click Create.

Install WordPress on Windows with Wamp

Download a copy of WordPress from WordPress.org. Extract the zip file and copy the folder WordPress. Go to the C: \ wamp \ www \ folder and paste WordPress there. You can rename a folder WordPress any name you want, for example mytestsite, mybeginnersite, etc. Now, open a Web page and navigate to: http://localhost/mytestsite/

The application will notify you that WordPress can be found on the file wp-config.php. Click on the button to create a profile (Create a Configuration File) to create it.

On the next screen, you must provide information for your database. It will be the name of the database the name of your choice for example test_db. The default will be the user name of your database is the foundation, and there will be a password, you can leave it blank for without password. Click on the submit button and WordPress will create the profile for you.

And then you will see a screen telling you that WordPress has created the profile successfully, and you can continue the installation. Click the Install button Run Install.

On the next screen, fill the form, you must provide the address of your local site, and choose a user name and password for the supervisor, and provide an email address for contact. In the end click on the install button.

Congratulation! :-) WordPress has installed now create database tables quickly. Once you're done, you'll see a message the installation was successful. And then you can go to a blog of your own by clicking on the login button.

Enjoy local Blogging!
